#  Análisis de Mercado: Alura Store Latam

## Descripción del Proyecto
Este proyecto es un análisis de datos enfocado en evaluar el rendimiento de cuatro tiendas de comercio electrónico en Latinoamérica. El objetivo principal es analizar diversas métricas financieras y operativas para recomendar a un inversionista (el Sr. Juan) cuál de las cuatro tiendas representa la mejor oportunidad de negocio.

## Objetivos
* Consolidar y limpiar bases de datos de diferentes sucursales.
* Calcular la facturación total y la cuota de mercado de cada tienda.
* Identificar las categorías de productos más rentables.
* Analizar los costos logísticos (envío) y la satisfacción del cliente (calificaciones).
* Proveer una recomendación final respaldada por datos.

## Tecnologías Utilizadas
* **Python** (Lenguaje principal)
* **Pandas** (Manipulación y análisis de datos)
* **Matplotlib** (Visualización de datos)
* **Google Colab / Jupyter Notebook** (Entorno de desarrollo)

## Conclusión y Recomendación Final

Tras analizar los datos, **se recomienda enfáticamente proceder con la Tienda 1**. 

**Justificación:**
1. **Dominio del Mercado:** Es la tienda con mayor facturación total ($1,150,880,400.00), garantizando un flujo de caja robusto.
2. **Equilibrio Operativo:** A pesar de su volumen masivo de ventas, mantiene una calificación promedio de **3.98 estrellas**, demostrando escalabilidad sin sacrificar la experiencia del cliente.
3. **Mix de Productos Estratégico:** Lidera en la venta de productos tecnológicos, lo cual optimiza el margen de beneficio por transacción.
